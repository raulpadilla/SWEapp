Contents list

When you first run an Siebel Systems - Interactive Selling application, a list of topics about the products available for selection and/or configuration is displayed. (This collection of topics is referred to as the Contents list.) Clicking a topic will expand or collapse a list of topics, or if the topic is linked to a product page, the page is displayed in the Inputs area.


Inputs area

When you click on a topic in the Contents list that is linked to an Inputs page, the page will be displayed in this area.

An Inputs page may contain several types of controls that allow you to specify the type of product you need. Each time you make a selection, the Results area provides immediate feedback (see Results area for details). For information about the controls available in the Siebel Systems - Interactive Selling application, click on a topic under Selection Controls in this help.


Results area

As you make selections in the Inputs area, you will get immediate feedback on whether a selection is part of a valid product configuration. (A valid configuration refers to the set of features that represent an available product.) This feedback appears in the Results area. If you make a selection that is not compatible with previous selections, the Results area will display suggestions that will help you make an appropriate choice. While you are making your requirement selections, this area may also display upgrades, promotions, and other beneficial product details.

Once you have selected all of the necessary requirements, the Results area displays information about the product that matches the requirement set.


Contents list button

If the Contents list is not visible, click on the button (commonly named Contents List or Products List) to display the list of products and topics provided in this Siebel Systems - Interactive Selling application.


Help button

Click on the Help button to display this information.


About button

Click on the About button to display information about the company whose Siebel Systems - Interactive Selling application you are using.


Single-selection group

This selection control allows you to select one item from the list. Click an item to select or remove the selection.


Single-selection list

You can select one item from this list. Use the scroll bar to display any items not currently visible. Click an item to select it. To change the selection, click the item again or click another item.


Single-selection dropdown list

Click the drop arrow next to this selection control to display the list of available items. Use the scroll bar to display any items not currently visible in the list. Click an item to select it. To change the selection, click the item again or click another item.


Multiple-selection group

You can select one or more items in this type of selection control. Click the box next to the item to select it. To change the selection, click the checked box.
