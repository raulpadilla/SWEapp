Siebel 서버/웹 서버와 UI(사용자 인터페이스) 간의 데이터 상호 작용은 스크립트를 통해 처리됩니다. 이를 위해서는 웹 브라우저에 JavaScript를 사용하도록 설정해야 합니다.

Internet Explorer에서 JavaScript를 설정하려면 다음과 같이 하십시오.

[도구] > [인터넷 옵션] > [보안] > [사용자 정의 수준]을 선택한 후 "Active 스크립트"를 "사용"으로 설정합니다(IE 버전 5.0의 경우).

[도구] > [인터넷 옵션] > [보안] > [사용자 정의 수준]을 선택한 후 "Java 애플릿 스크립트"를 "사용"으로 설정합니다(IE 버전 5.0의 경우).

다른 브라우저를 사용하는 경우에도 해당 절차에 따라 JavaScript를 사용하도록 설정하여 Java 애플릿에서 JavaScript를 처리할 수 있게 하십시오.
