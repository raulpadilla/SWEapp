Os scripts são utilizados para gerenciar a interação de dados entre o Siebel server/Servidor da Web e a interface do usuário. Para isso, o JavaScript deve ser ativado no navegador da Web

No Internet Explorer, faça o seguinte:

Escolha Ferramentas> Opções da Internet> Segurança> Diversos> defina ‘Scripts ativo’ como prompt ou ativo.

Escolha Ferramentas> Opções da Internet> Segurança> Diversos> defina ‘Script de Java Applets’ como prompt ou ativo.

Para outros navegadores, siga instruções adequadas para ativar o uso do JavaScript e para permitir que applets do Java interajam com JavaScript.
