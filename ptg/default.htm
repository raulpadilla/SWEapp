O scripting é utilizado para gerir interacções de dados entre o Siebel Server/servidor Web e a Interface do utilizador. É necessário activar o JavaScript no Web browser

Para configurar esta opção no Internet Explorer, execute o seguinte procedimento.

Seleccione Ferramentas > Opções da Internet > Segurança > Diversos > e active a opção “Processamento de scripts activo”

Seleccione Ferramentas > Opções da Internet > Segurança > Diversos > e active a opção “Processamento de scripts de aplicações Java”

Para outros browsers, siga as instruções equivalentes para activar a utilização de JavaScript e para permitir que aplicações Java interajam com JavaScript.
