Zur Verwaltung von Dateninteraktionen zwischen dem Siebel Server/Web-Server und der Benutzerschnittstelle werden Skripte verwendet. Daher muss JavaScript im Web-Browser aktiviert sein.

Im Internet Explorer können Sie JavaScript folgendermaßen aktivieren.

Wählen Sie Extras > Internetoptionen > Sicherheit > Internet, klicken Sie auf die Schaltfläche „Stufe anpassen ...“, und setzen Sie „Active Scripting“ auf „Aktivieren“ oder „Eingabeaufforderung“

Wählen Sie Extras > Internetoptionen > Sicherheit > Internet, klicken Sie auf die Schaltfläche „Stufe anpassen ...“, und setzen Sie „Scripting von Java-Applets“ auf „Aktivieren“ oder „Eingabeaufforderung“

Bei anderen Browsern folgen Sie zur Aktivierung von JavaScript und der Interaktion zwischen Java-Applets und JavaScript den entsprechenden Anweisungen.
