Download and Installation

The page you are trying to view contains Ektron's eWebEditPro editor. It will appear within your browser. It allows you to enter content for web pages as easily as using a word processor.

Before you can use eWebEditPro, it must be downloaded into your browser. When you click the Install Now button at the bottom of this page, eWebEditPro will be downloaded and installed. This process may take several minutes depending on the speed of your network connection. Once downloaded, eWebEditPro will not need to download again unless upgrading to a newer version.

You must have authorization to install programs on your computer.

By default, Internet Explorer prompts you for approval before beginning any installation on your computer. A screen similar to the one below will appear. Please click Run to proceed.

By default, Mozilla prompts you for approval before downloading any installation on your computer. A screen similar to the one below will appear. Please click OK to proceed.

Once it downloaded, you can double click on the ewebeditproclient.exe to start the installation.

A screen similar to the one below will appear. This is to select the language for the installation. Please select your desired language and click OK to proceed.

Then click Next or Install until the installation is finished.

When the installation is finished, please click Finish. This page will closed and the original page will be refreshed.



Click on Install Ektron eWebEditPro to proceed, or close this window if you do not want to install eWebEditPro.
