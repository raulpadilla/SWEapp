This help page provides information on how to use the various controls that might appear in your Rainmaker Sales application. The configuration application resides within the browser page you are viewing. While there may be other controls available within this browser page, this help file supports the configuration application controls only.

The types of things you may see in a configuration area are divided into the following categories: information display areas, buttons, and selection controls. Click a topic from the list at left to get information about an area or control.
