Updating Your User Profile

Contents

You use the User Profile form to keep your profile information up to date.

To update your user profile
  1. Navigate to the User Profile form.
  2. Click Edit.
    The User Profile form appears.
  3. To change your password, click Change Password, fill in the required fields, and then click OK.
  4. In the User Profile form, make any other changes.
  5. Click Save.
