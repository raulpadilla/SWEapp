Scripting จะใช้ในการจัดการเกี่ยวกับการโต้ตอบข้อมูลระหว่างเซิร์ฟเวอร์/เว็บเซิร์ฟเวอร์ของ Siebel และอินเตอร์เฟซผู้ใช้ (UI) ซึ่งต้องใช้ JavaScript เพื่อให้สามารถใช้งานเว็บเบราเซอร์ได้

ในการตั้งค่าเพื่อใช้งานใน Internet Explorer โปรดดำเนินการดังต่อไปนี้

เลือก Tools > Internet Options > Security > Miscellaneous > ตั้งค่า "Active Scripting" เพื่อให้พรอมต์หรือใช้งาน

เลือก Tools > Internet Options > Security > Miscellaneous > ตั้งค่า "Scripting of Java Applets" เพื่อให้พรอมต์หรือใช้งาน

โปรดทำตามคำสั่งเดียวกันนี้กับเบราเซอร์ชนิดอื่นๆ เพื่อให้สามารถใช้งาน JavaScript และเพื่อให้ใช้ Java applet ในการโต้ตอบข้อมูลกับ JavaScript ได้
