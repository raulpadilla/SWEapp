El proceso de registro en este sitio web está diseñado para dirigirse tanto a usuarios anónimos como registrados. Puede visitar algunas áreas del sitio de forma anónima; por ejemplo, puede realizar búsquedas en las Preguntas más frecuentes sin registrarse. Los usuarios registrados disponen de un uso más amplio del sitio web.

Autorregistrarse

Para autorregistrarse
  1. Vaya a la página inicial.
  2. En el formulario de inicio de sesión, haga clic en el hiperenlace Nuevo usuario y siga las instrucciones que aparecen en pantalla.
    Cuando haya completado el proceso de registro, aparecerá la página inicial.

Acceder al sitio web sin registrarse

Para obtener acceso al sitio web sin registrarse
  1. Vaya al sitio web.
  2. Examine el sitio web.
