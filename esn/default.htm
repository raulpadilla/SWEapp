Las secuencias de comandos se utilizan para gestionar las interacciones de datos entre Siebel server, el servidor Web y la interfaz de usuario. Es necesario tener JavaScript activado en el explorador Web.

Lleve a cabo los siguientes pasos para configurarlo en Internet Explorer:

Seleccione Herramientas > Opciones de Internet > Seguridad > Misceláneo y establezca la opción “Pedir datos” o “Activar” de “Secuencias de comandos Active X”.

Seleccione Herramientas > Opciones de Internet > Seguridad > Misceláneo y establezca la opción “Automatización de los subprogramas de Java” en “Activar” o “Pedir datos”.

Para configurar JavaScript en otros exploradores siga las instrucciones correspondientes y permita que los subprogramas de Java interactúen con JavaScript.
