Il linguaggio di scripting viene utilizzato per la gestione delle interazioni dei dati tra il server Siebel/server Web e l'interfaccia utente. Questo linguaggio richiede l'attivazione di JavaScript nel browser Web.

Per l'impostazione dello scripting in Internet Explorer, eseguire le operazioni riportate di seguito.

Scegliere Strumenti > Opzioni Internet > Protezione > Varie >, imposta "Active Scripting" su Attiva o Chiedi conferma

Scegliere Strumenti > Opzioni Internet > Protezione > Varie >, imposta "Scripting delle applet Java" su Attiva o Chiedi conferma

Per altri browser, seguire le istruzioni equivalenti per l'abilitazione di JavaScript e per fare in modo che le applet Java interagiscano con JavaScript.
